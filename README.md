# MohitKhare .NET Utilities

Developer utilities and tools for .NET projects.

## Overview

A collection of developer utilities including token counting, string helpers, and common development tools.

- **Homepage**: [https://mohitkhare.me](https://mohitkhare.me)
- **Documentation**: [https://mohitkhare.me/docs](https://mohitkhare.me/docs)
- **Portfolio**: [https://mohitkhare.me/projects](https://mohitkhare.me/projects)

## Installation

```bash
dotnet add package MohitKhare
```

## Quick Start

```csharp
using MohitKhare;

Console.WriteLine($"MohitKhare v{Info.Version}");
Console.WriteLine($"Homepage: {Info.BaseUrl}");
```

## Features

- Developer utility functions
- Token counting helpers
- String processing tools
- Common .NET extensions
